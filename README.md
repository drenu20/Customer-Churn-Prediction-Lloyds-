# Customer-Churn-Prediction-Lloyds
This project predicts customer churn for Lloyds Banking Group using machine learning techniques. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and evaluation. The goal is to identify customers at risk of leaving and provide actionable insights for retention strategies. 

Project Overview:

Customer churn refers to customers stopping the use of a company's products or services. Predicting churn helps businesses proactively retain valuable customers. In this project, we analyze customer data, build predictive models, and evaluate their performance.

Features:

Data cleaning and preprocessing
Exploratory Data Analysis (EDA) with visualizations
Feature engineering and selection
Machine Learning models: Logistic Regression, Random Forest, XGBoost
Model evaluation using accuracy, precision, recall, and F1-score
Insights to support customer retention strategies

Dataset:
The dataset used simulates customer information from Lloyds Banking Group. It includes customer demographics, account details, and service usage metrics.

Technologies Used:
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
